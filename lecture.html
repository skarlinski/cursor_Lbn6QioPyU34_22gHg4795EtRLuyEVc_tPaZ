Deep Dive

Deep Dive

Choosing Models

  • You should never reach the full context of your model.
  • Smallest context window ≈ 200k tokens — about 700 pages / 20k lines of code
  • Break tasks with /summarize
  • Fast models vs thinking models — token difference
  • Models have different prices — gemini < openai < claude
